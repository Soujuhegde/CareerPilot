## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **AI Career Roadmap** | Get a personalized, step-by-step career path generated by AI based on your industry and experience |
| 📝 **AI Cover Letter Generator** | Generate tailored, professional cover letters for any job in seconds |
| 🎤 **Mock Interview Practice** | Practice with AI-generated role-specific questions and track your performance over time |
| 📊 **ATS Score Analyzer** | Upload your resume and get instant feedback on how well it matches Applicant Tracking Systems |

| 📈 **Industry Insights & Trends** | Real-time salary ranges, demand trends, and skill gap analysis for your industry |
| 📚 **Course Recommendations** | AI-curated course suggestions to close your skill gaps and advance your career |
| 🏗️ **Resume Builder** | Build ATS-friendly resumes with a live markdown editor and export to PDF |
| 🔍 **Recruiter Dashboard** | Recruiters can search, filter, and shortlist top candidates with AI smart matching |

## 🚀 Getting Started

### Prerequisites

- Node.js `>= 18`
- A [Clerk](https://clerk.com/) account
- A [Neon](https://neon.tech/) PostgreSQL database
- A [Google Gemini API](https://ai.google.dev/) key

### 1. Clone the Repository

```bash
git clone https://github.com/Soujuhegde/CareerPilot.git
cd CareerPilot
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Environment Variables

Create a `.env` file in the root directory:

```env
# Database (Neon PostgreSQL)
DATABASE_URL="postgresql://..."

# Clerk Authentication
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/industry-insights
NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/industry-insights

# Google Gemini AI
GEMINI_API_KEY=AIza...
```

### 4. Push Database Schema

```bash
npx prisma db push
```

### 5. Run the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## 🌐 Deployment

This project is deployed on **Vercel**. To deploy your own:

1. Fork this repository
2. Import the project into [Vercel](https://vercel.com/)
3. Add all environment variables in the Vercel dashboard
4. Add your Vercel domain to **Clerk's allowed domains**
5. Deploy!
